# dotfiles

Currently, this repo versions the following dotfile packages.

- `alacritty`
- `zsh`
- `fastfetch`

## Dependencies

Make sure you have the following installed.

- [`CaskaydiaMono Nerd Font`](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/CascadiaMono.zip)
- `zsh` autosuggest and autocomplete plugins. See [this](https://gist.github.com/n1snt/454b879b8f0b7995740ae04c5fb5b7df) guide
